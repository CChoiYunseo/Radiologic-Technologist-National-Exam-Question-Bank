# Detailed Question Language Rulebook

## 법에 맞게

- Scope: 공공 언어, 한글 우선 표기, 단위 표기, 전문어 병기
- Source pages: 5, 7, 8, 9, 10, 11, 12, 13

### QL-001 한글 우선 표기와 원어 병기

- Requirement: 문항은 한글을 우선하고, 뜻 전달에 필요한 경우에만 괄호 안에 한자·영문·약어를 병기한다.
- Check: 전문 용어 또는 약어가 처음 나올 때 한글 설명 없이 외국 문자만 제시하지 않았는지 확인한다.
- Severity: error

### QL-002 단위 표기

- Requirement: 수치와 단위 기호 사이의 띄어쓰기 원칙을 지키되, 한글 단위 기호, ℃, % 등은 국시원 표기 관례에 맞춘다.
- Check: 수치+단위 표기가 일관적인지 확인한다.
- Severity: warning

## 정확하게

- Scope: 어문 규범, 어휘 선택, 조사, 피동, 사동, 시제, 병렬, 호응, 어순
- Source pages: 14, 15, 16, 17, 18, 19, 20, 21

### QL-003 어문 규범 준수

- Requirement: 한글 맞춤법, 표준어, 외래어 표기, 로마자 표기, 문장 부호 규정을 따른다.
- Check: 비표준어, 틀린 두음법칙, 사이시옷, 띄어쓰기, 문장부호 오류가 없는지 확인한다.
- Severity: error

### QL-004 정확한 어휘 선택

- Requirement: 출제 의도에 맞는 단어를 선택하고, 유사어를 혼동하지 않는다.
- Check: 완전/완료, 전부/전체, 시행/실시/실행 등 뜻이 다른 단어를 구분했는지 확인한다.
- Severity: warning

### QL-005 조사 선택

- Requirement: 조사는 앞말과 서술어의 관계를 정확히 드러내도록 선택한다.
- Check: 사람/동물 대상에는 '에게', 장소/사물에는 '에'처럼 대상과 의미에 맞는 조사를 썼는지 확인한다.
- Severity: warning

### QL-006 피동 표현 절제

- Requirement: 불필요한 피동, '-에 의하여', '-어지다', '-되다' 중첩을 피하고 능동문을 우선한다.
- Check: 행위 주체와 목적어가 분명한데 피동으로 돌려 쓴 표현이 없는지 확인한다.
- Severity: warning

### QL-007 사동 표현 절제

- Requirement: '-시키다'는 실제로 누군가에게 하게 만드는 경우에만 쓰고, 자체로 사동 의미가 있는 단어에는 붙이지 않는다.
- Check: 향상시키다, 개선시키다, 감소시키다처럼 불필요한 사동이 없는지 확인한다.
- Severity: warning

### QL-008 시제 일관성

- Requirement: 사례형 문항에서 과거 병력, 현재 증상, 앞으로의 처치 판단 시제가 일관되어야 한다.
- Check: 문항이 현재 판단을 요구하는데 과거형만 나열해 의미가 어색하지 않은지 확인한다.
- Severity: warning

### QL-009 병렬 구조 일치

- Requirement: 나열된 보기와 지시문의 품사, 문장 구조, 의미 층위가 서로 맞아야 한다.
- Check: 보기 일부는 명사구, 일부는 문장처럼 섞여 있지 않은지 확인한다.
- Severity: warning

### QL-010 호응

- Requirement: 주어-서술어, 목적어-서술어, 부사어-서술어가 문법적으로 호응해야 한다.
- Check: 생략된 주어/목적어 때문에 누가 무엇을 하는지 불명확하지 않은지 확인한다.
- Severity: error

### QL-011 어순

- Requirement: 긴 수식어나 부사구 때문에 의미가 흐려지지 않도록 가까이 있어야 할 성분은 가까이 둔다.
- Check: 수식어가 어떤 단어를 꾸미는지 명확한지 확인한다.
- Severity: warning

## 명료하게

- Scope: 중의적 표현, 부적절한 수식, 지시문과 답지의 조화
- Source pages: 61, 62, 63, 64, 65, 66, 67, 68

### QL-012 중의적 표현 금지

- Requirement: 하나의 문장이나 어구가 둘 이상으로 해석되면 안 된다.
- Check: 행위 주체, 대상, 수식 범위, 시간/조건이 한 가지로 해석되는지 확인한다.
- Severity: error

### QL-013 수식 관계 명확화

- Requirement: 수식어와 피수식어의 관계가 구조상 조화해야 한다.
- Check: 형용사나 관형어가 엉뚱한 명사나 결과를 꾸미지 않는지 확인한다.
- Severity: warning

### QL-014 지시문 내용과 표현 일치

- Requirement: 지시문은 실제로 묻는 대상과 선택 기준을 분명히 표현해야 한다.
- Check: '설명으로 옳은 것은?'이 실제로 설명형 보기를 요구하는지 확인한다.
- Severity: error

### QL-015 지시문과 답지 조화

- Requirement: 답지에 둘 이상의 항목을 묶거나 짝지어 제시하면 지시문도 그 구조를 반영해야 한다.
- Check: 보기 형식이 조합형/연결형이면 '옳게 짝지은 것은' 등으로 묻는지 확인한다.
- Severity: error

## 충분하게

- Scope: 문항 이해에 필요한 주어, 목적어, 행위자, 조건, 대상 보충
- Source pages: 79, 80, 81, 82, 83

### QL-016 필수 정보 보충

- Requirement: 수험생이 지시문과 답지를 이해하는 데 필요한 행위자, 대상, 조건, 기준이 빠지면 안 된다.
- Check: 누가, 무엇을, 누구에게, 어떤 조건에서, 무엇으로 판단하는지 확인한다.
- Severity: error

## 자연스럽게

- Scope: 국어 기본 문형, 외국어투, 명사문 회피
- Source pages: 84, 85, 86, 87, 88, 89, 90, 91

### QL-017 국어 기본 문형

- Requirement: 주어, 목적어, 부사어, 서술어를 서술어 성질에 맞춰 배치한다.
- Check: 명사 나열이나 조사 생략 때문에 국어 문장성이 떨어지지 않는지 확인한다.
- Severity: warning

### QL-018 외국어투 회피

- Requirement: 영어식 수동태, '가지다', 명사문식 표현을 자연스러운 국어 표현으로 바꾼다.
- Check: 가지다, -에 의해, -에 대한, 명사+목적 등 외국어투가 불필요하게 쓰이지 않았는지 확인한다.
- Severity: warning

## 쉽게

- Scope: 어려운 전문어, 일반어, 약어를 이해하기 쉽게 표현
- Source pages: 95, 96, 97, 98, 99

### QL-019 쉬운 말 우선

- Requirement: 일반화되지 않은 약어와 어려운 전문어는 가능한 쉬운 말 또는 한글 설명으로 바꾼다.
- Check: 전문어를 써야 하면 한글 설명, 원어, 약어를 일관되게 제시한다.
- Severity: warning

## 간결하게

- Scope: 문제 해결에 불필요한 어구와 중복 표현 제거
- Source pages: 100, 101, 102, 103, 104, 105, 106

### QL-020 불필요한 정보 제거

- Requirement: 문제 해결에 필요하지 않은 배경, 중복 수식, 장황한 표현은 제거한다.
- Check: 삭제해도 정답 판단에 영향이 없는 문구가 없는지 확인한다.
- Severity: warning

## 그 밖에

- Scope: 고압적, 비격식, 비객관적, 외국어 남용, 비일관적 표현 방지
- Source pages: 107, 108, 109, 110, 111, 112, 113, 114

### QL-021 고압적 표현 금지

- Requirement: 수험자나 환자에게 감정적, 위압적, 명령조로 읽히는 표현을 피한다.
- Check: 지시하였다, 명령하였다 등 고압적 표현이 필요한 맥락인지 확인한다.
- Severity: warning

### QL-022 비격식 표현 금지

- Requirement: 공식 시험 문항에서는 일상어보다 공식 명칭을 우선한다.
- Check: 해썹 등 비격식 명칭이 공식 명칭과 병기되거나 대체되었는지 확인한다.
- Severity: warning

### QL-023 비객관적 표현 금지

- Requirement: 부정적 사건이나 지역·집단을 불필요하게 특정해 편견을 만들지 않는다.
- Check: 특정 지역, 집단, 개인을 근거 없이 부정적으로 드러내지 않았는지 확인한다.
- Severity: error

### QL-024 외국어 남용 금지

- Requirement: 대체 가능한 우리말이 있으면 외국어만 단독으로 쓰지 않는다.
- Check: 외국어/외래어가 일반 전문어인지, 대체어가 있는지 확인한다.
- Severity: warning

### QL-025 보기 표현 일관성

- Requirement: 한 문항 안에서 보기의 용어, 문체, 문장 형식, 표기 방식이 일관되어야 한다.
- Check: 정답 보기만 표현 형식이 다르거나 더 구체적이지 않은지 확인한다.
- Severity: warning
