Project Presentation

방사선사 국가고시 AI 문제은행 구축

공식 출제범위, 문항 설계 기준, 국문법 기준, 검수 완료 출제비중, 전공 근거 자료를 연결해 검증 가능한 문제은행으로 구축한다.

Overview

프로젝트 개요

  • 방사선사 국가고시 전체 범위를 대상으로 하는 AI 문제은행 시스템
  • 텍스트 기반 이론·법규·응용 문항과 영상·장비 기반 실기 판단형 문항을 함께 고려
  • 단순 AI 생성이 아니라 공식 기준, 출제비중, 근거 자료를 반영하는 구조
  • 최종 목표는 근거 검색, 문항 생성, 자동 검증, 전문가 검수, 풀이 서비스 연결
Why

왜 필요한가

학습 수요

방사선사 국가고시는 범위가 넓고 단원별 반복 학습용 문항 수요가 크다.

기존 방식의 문제

기출·모의고사 중심 접근은 저작권, 최신성, 범위 편중 문제가 발생할 수 있다.

AI 활용 가능성

단원, 난이도, 유형을 지정해 다양한 문항을 빠르게 생성할 수 있다.

필수 통제

의료·보건 시험 문항은 근거성, 정답 유일성, 법규 최신성, 표현 명확성이 중요하다.

Architecture

전체 시스템 흐름

기준 자료출제범위, 학습목표, 문항 설계, 국문법 기준을 구조화한다.
전공 자료PDF 텍스트와 표를 추출하고 검색 가능한 단위로 나눈다.
근거 검색·생성출제범위에 맞는 근거를 찾고 AI가 문항 초안을 생성한다.
검증·저장자동 검증과 전문가 검수를 거쳐 문제은행에 저장한다.
분류 정보교시, 과목, 영역, 세부영역, 난이도, 문항 유형을 모든 문항에 부여한다.
근거 정보근거 문단, 출처 파일, 페이지 또는 법령 기준일을 보존한다.
검수자동 검증은 1차 필터이며 최종 승인은 검수 과정을 거친다.

자료 관리 방식

문서 정보와 출처는 별도로 관리하고, 의미 기반 검색용 색인을 만들어 문항 생성 시 근거를 찾는다.

Roadmap

다음 개발 단계

서비스관리자 검수 화면, 풀이 화면, 오답노트 구현

영상형 문항은 이미지 정합성·비식별·정답 유일성 검토 기준을 먼저 설계한 뒤 제한적으로 확장한다.

Risk

주요 한계와 대응 방향: 1·2교시

저작권기출·복원·시중 모의고사 원문은 복제 위험이 있어 근거 자료로 직접 등록하지 않는다.
AI 생성 오류근거에 없는 설명, 수치, 법령 조항이 생성될 수 있어 출처와 페이지를 남긴다.
정답 유일성정답이 2개 이상으로 해석되거나 정답 단서가 생기지 않도록 자동 검증과 전문가 검수를 둔다.
법규 최신성의료관계법규는 개정 가능성이 높아 법령 기준일과 조문 단위 관리가 필요하다.
문서 추출 품질스캔본과 표 구조 오류가 검색·생성 품질을 흔들 수 있어 추출 상태를 별도로 확인한다.
검색 품질전공 자료가 부족하거나 잘못 분할되면 근거 검색과 생성 결과가 함께 흔들릴 수 있다.
Visual Items

3교시 영상·장비 기반 문항의 추가 고려사항

3교시에서 AI가 영상을 만들고 그 이미지로 문제를 출제하려면, 단순 문항 생성보다 훨씬 강한 통제가 필요하다.

비식별·권한환자 정보, 병원명, 촬영일, 저작권이 남아 있으면 교육용 활용이 어렵다.
이미지 정합성AI 생성 영상은 그럴듯해 보여도 해부학, 병변 위치, 영상물리 오류가 생길 수 있다.
정답 유일성영상이 폐렴·흉막삼출처럼 여러 해석을 만들면 5지선다 문항으로 사용할 수 없다.
범위 관리희귀 질환이나 전문의 시험 수준으로 넘어가지 않도록 출제범위와 학습목표를 강제해야 한다.
검수 부담이미지와 문항을 함께 검토해야 하므로 교수, 방사선사, 전문의 검수 부담이 커진다.
자료 편향흉부 X-ray는 많지만 PET, SPECT, 특수 장비 영상은 부족해 과목별 균형이 깨질 수 있다.

따라서 1·2교시는 RAG 기반 문제은행을 먼저 구축하고, 3교시 이미지 생성은 공개 의료영상 활용 또는 별도 연구 과제로 단계화하는 것이 현실적이다.

Conclusion

결론

이 프로젝트의 핵심은 문제를 많이 만드는 것이 아니라, 근거 있고 검증 가능한 문제를 만드는 것이다.

  • 현재까지 출제범위, 출제비중, 학습목표, 검증 기준을 시스템의 뼈대로 고정
  • 다음 단계는 관리자 선택 화면, 전공 자료 기반 검색 체계, 문항 생성 기능 구축
  • 저작권, 법규 최신성, 정답 오류, 문서 추출 품질, 검색 품질, 의료영상 검증은 지속 관리해야 할 핵심 리스크
